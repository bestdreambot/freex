## Что такое OpenFugu

Это **"одна модель, которая командует всеми"** — умный диспетчер, который:
- Берёт твой запрос
- Решает, какую модель (локальную или облачную) лучше использовать
- Возвращает лучший результат

Открытая реализация закрытой технологии Sakana Fugu.

---

## Основные возможности


- OpenAI-совместимый сервер (`/v1/chat/completions`)
- Режим Fugu (быстрый routing)
- Режим Fugu-Ultra (агентный workflow / DAG)
- Возможность обучать свой оркестратор
- Работает с локальными моделями и API (litellm)

## 🚀 Что такое OpenFugu

Открытая реализация **Sakana Fugu** — «одна модель, которая командует всеми».

Это **умный оркестратор**, который:
- Принимает запрос
- Анализирует задачу
- Выбирает лучшую модель из пула (локальные + облачные)
- Возвращает оптимальный результат

Поддерживает два режима:
- **Fugu** — быстрый routing
- **Fugu-Ultra** — полноценный агентный workflow (DAG)

# Стартовый запрос для нового чата / нового ИИ

Скопируй и вставь как первое сообщение в новом чате с любым ИИ-агентом:

```
Продолжаем FreeX. Сначала прочитай PROJECT_STATUS.md и AI_RESUME.md.
Затем восстанови контекст, сообщи текущее состояние проекта, последний
завершённый шаг и предложи следующий. Не повторяй уже выполненную работу.
Ничего не меняй без отдельной задачи.
```

Перед этим при необходимости запусти восстановление состояния:
- двойной клик по `Resume FreeX.cmd`, или
- в VS Code: `Ctrl+Shift+B` (задача `FreeX: Resume Work`), или
- вручную: `powershell -File scripts/resume-freex.ps1`

## Команда «ок» — продолжение внутри уже идущего чата

Если чат уже восстановлен и работа шла, а потом прервалась (обрыв связи,
новая сессия в той же беседе) — достаточно написать одно слово:

```
ок
```

(распознаётся и `OK`, `Ok`, `okay` — без лишних слов, пробелы по краям не
считаются). Агент обязан прочитать `AI_RESUME.md` (раздел «Текущий шаг») и
`PROJECT_STATUS.md`, определить последний завершённый шаг и незавершённый,
не повторять завершённое и продолжить именно с места остановки. Полный
протокол — в `AI_RESUME.md` → «Команда «ок»» и в `agents/claude.md`.

«ок» не даёт разрешения на удаление данных, `git reset`, force push,
раскрытие секретов, установку программ или изменение настроек Windows —
только на продолжение уже утверждённой задачи безопасными средствами.

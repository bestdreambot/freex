# FreeX 8.1

Личная мобильная среда для работы с несколькими ИИ сразу — чаты, заметки, папки, режим «Конференция» (все доступные модели отвечают по очереди).

Полная документация проекта — в [start.md](start.md). Начни с него.

## Структура

- `index.html` — разметка, подключает `js/app.js`
- `js/` — модульная логика приложения (точка входа: `js/app.js`)
- `api/chat.js` — серверный прокси провайдеров (Vercel Function)
- `style.css` — все стили

## Провайдеры

DeepSeek, GPT-4o и DeepSeek-R1 (GitHub Models), Gemini 2.5 Flash, Llama 3.3 (Groq, Cloudflare Workers AI). DeepSeek работает и без личного ключа — сервер использует `DEEPSEEK_API_KEY` из переменных окружения.

## Развёртывание

Проект развёрнут на Vercel из ветки `main`. Секреты (`DEEPSEEK_API_KEY` и т.д.) задаются только в Vercel Environment Variables — никогда не коммитятся в репозиторий.

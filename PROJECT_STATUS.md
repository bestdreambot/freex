## Основные компоненты проекта
- `index.html` — только разметка, подключает `js/app.js`
- `js/app.js` — точка входа: инициализация состояния, обработчики событий, `sendMessage`
- `js/config.js` — модели, ключи localStorage, версия приложения (`APP_VERSION`)
- `js/state.js` — единое состояние приложения (event-based store)
- `js/db.js` — Dexie/IndexedDB (заметки)
- `js/api.js` — вызов `/api/chat` с таймаутом на запрос
- `js/chat.js` — чаты, Конференция, рендер сообщений
- `js/notes.js` — заметки: CRUD, поиск, теги, автосохранение черновика
- `js/sidebar.js` — папки и список чатов
- `js/github.js` — деплой файлов в репозиторий через GitHub API
- `js/ui.js` — темы, sidebar toggle, toast-уведомления
- `js/utils.js` — общие хелперы
- `api/chat.js` — серверный прокси всех провайдеров (Vercel Function)

## 4. Через какие сложности прошли (и как решили)
1. **Регистр путей** — Index.html/API с большой буквы Vercel не видел, плодились дубли. Решение: выпадающий список путей в деплое — только маленькие буквы.
2. **CORS** — прямые вызовы из браузера блокировались, работал только DeepSeek. Решение: прокси на Vercel.
3. **DeepSeek-R1 — пустой ответ / `</think>` в тексте.** Решение: прокси берёт всё после `</think>`; таймаут поднят до 60 сек.
4. **Gemini — ложная заглушка при STOP.** Решение: собирать текст из всех parts, отдавать при наличии.
5. **Лимит токенов GPT-4o (8000).** Решение: история ограничена 12 сообщениями.
6. **Groq — пустой ответ.** Решение: явный max_tokens.
7. **Платные модели-пустышки** (Grok xAI, Grok OpenRouter, Cerebras, Mistral) — убраны: у них нет реального бесплатного доступа.
